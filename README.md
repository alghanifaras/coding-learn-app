# Coding Learn App

> A small learning platform combining a Laravel backend API and a Next.js frontend.

This repository contains two main folders:

- `backend/` — Laravel API (authentication with Sanctum, models, migrations, seeders)
- `frontend/` — Next.js 14 app (React components, pages, and API client)

See the documentation in the `docs/` folder for details:

- [docs/backend.md](docs/backend.md)
- [docs/frontend.md](docs/frontend.md)
- [docs/api.md](docs/api.md)
- [docs/development.md](docs/development.md)

Created files provide quick start instructions, API endpoint details, and component overviews to help contributors and maintainers get started.
